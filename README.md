# markdown-cv

A curriculum vitae maintained in plain text and rendered to HTML and PDF using CSS.

For more details, see the [project page](http://elipapa.github.io/markdown-cv)

## Build it locally and print a PDF

1. install [jekyll](https://jekyllrb.com/docs/installation/), run `gem install bundler jekyll` from the command line.
1. Type `jekyll serve` to render your CV at http://localhost:4000.
1. To print a PDF, press <kbd>âŒ˜</kbd> + <kbd>p</kbd>.
